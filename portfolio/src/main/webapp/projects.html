Project 1

Menu translator
With Python, I and three other people worked together during ShellHacks(Hackathon) that would allow for an image to be to be read in and translated to any language. We created a GUI that would allow the user to select the language to translate to and image to be translated. This is very helpful with menus and/or instructions that are in a language the user does not understand

Project 2

Project Beats
With javascript, html, css and python, I and three of my friends created a website from scratch. We also implemented Magenta API, an API that allows us to to create our own music through machine learning networks that have been established. With the Magenta API we created a website that showcased who we are and showcased the music that was created through the api.

Project 3

Project Jerry
Personal project that implements arduino and python to create an RGB audio reactive lamp. With an arduino wifi module I connected the strips of LEDS to a python script the continuously updates the strips based on the frequency of the audio being listened to. The script allows for different light options (i.e. pulse, energy, free flow, spectrum). The future plans for this lamp is to add voice commands and more functionality and move it to a smaller compact place.
